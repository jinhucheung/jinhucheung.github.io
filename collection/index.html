收集一些好玩有趣的项目

GitHub 相关

  • GitHub Chart API: 为你的 GitHub 贡献度生成一个图片
  • GitHub Corners: “Fork me on GitHub” 漂亮标识图
  • GitHub Buttons: 生成 GitHub Follow, Star 按钮
  • Git History: 动态显示 Git Commit 历史的平台
  • GitHub Readme Stats: 在 Redmine 中动态生成 GitHub 统计信息
  • IssueHunt: 开源打赏平台，为解决 Issue 的人提供一笔奖金～
  • HelloGitHub: 分享、推广 GitHub 上有意思、高质量的项目
  • github-changelog-generator: 自动根据项目的 Issue, PR 等信息生成 change log
  • GitHub Style Guide: Ruby, Css, JavaScript style guide
  • GitHub Assets: 获取 GitHub 官方使用的资源，如 pinned-octocat

效率工具

  • Tomato Pie: 番茄工作法的 Chrome 插件～
  • 墨刀: 一个很棒的在线产品原型设计平台 :)
  • 石墨文档: 文档实时协同平台，来记录你的学习、工作吧～
  • 在线配色方案自动生成器: 挑选在不同亮度下的颜色
  • ProcessOn: 一个 UI 很漂亮的在线作图平台，支持流程图、思维导图等等
  • docsmall: 一个免费易用的在线图片压缩平台
  • favicon.io: 一个精致的网站图标生成平台
  • SVG Repo: Search, explore, edit and share open licensed SVG vectors
  • 思知：对外开放接口的中文知识图谱平台
  • unDraw: 免费精美的矢量图平台
  • Avataaars generator: 卡通头像生成工具
  • asdf: 多语言版本控制工具

DevOps

  • Docker for Rails development: 为 Rails 开发搭建 Docker 环境
  • Capistrano: 自动化部署工具，与 Rails 集成十分方便
  • Certbot: 自动签发 Let’s Encrypt SSL 证书的工具

让生活更加多彩～

  • 下厨房: 不知做什么菜时就上这儿吧 XD
  • Keep: 现在起开始记录跑步～
